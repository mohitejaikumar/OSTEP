# OSTEP

Homework solutions and notes for
[Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/),
organized by chapter.

## Contents

- [`Processes/`](Processes/README.md) — the `process-run.py` scheduling simulator homework.
- [`Process-API/`](Process-API/README.md) — `fork()`, `exec()`, `wait()`, pipes, and the UNIX process API.
- [`Direct-Execution/`](Direct-Execution/README.md) — measuring the cost of a system call and a context switch.

Each directory has its own `README.md` with the homework writeup, code, and
observed results.
